## Descripción:

En esta actividad se hace el análisis de un dataset proporcionado por una universidad que contiene los datos históricos referentes a sus solicitudes de aplicación recibidas, sus estudiantes admitidos,
sus estudiantes matriculados, el porcentaje de satisfacción de aquellos estudiantes matriculados, la tasa de retención de la universidad y por último la cantidad de estudiantes matriculados por facultad,
todos estos datos organizados por año.

## Objetivo:

El objetivo principal es facilitar el análisis de los datos proporcionados haciendo uso de gráficos dinámicos y representaciones visuales de la información. Esto permite:
- Observar la evolución de las ***solicitudes de admisión*** por año y término.
- Evaluar las ***tasas de retención*** y su tendencia histórica.
- Analizar la relación entre la ***satisfacción estudiantil*** y la ***retención***.
- Comparar el número de ***matriculados por facultad***.
- Filtrar dinámicamente por ***año*** y ***término académico***.

## Descripción del dataset

El archivo `university_student_data.csv` contiene las siguientes columnas:

| Columna | Descripción |
|----------|--------------|
| **Year** | Año académico. |
| **Term** | Término (Spring, Fall). |
| **Applications** | Número de solicitudes de admisión recibidas. |
| **Admitted** | Cantidad de estudiantes admitidos. |
| **Enrolled** | Número total de estudiantes que se matricularon. |
| **Retention Rate (%)** | Porcentaje de estudiantes que continuaron sus estudios al siguiente período. |
| **Student Satisfaction (%)** | Nivel promedio de satisfacción reportado por los estudiantes. |
| **Engineering Enrolled**, **Business Enrolled**, **Arts Enrolled**, **Science Enrolled** | Distribución de matriculados por facultad o área académica. |
